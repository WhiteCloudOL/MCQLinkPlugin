# MCQLinkPlugin

## 它能做什么？
1. 建立websocket客户端连接  
2. 收发事件消息  

## 支持版本  
1. release由 Java21 编译，支持Minecraft 1.21.X及以上任意版本  
2. 理论可以支持Java8以上版本，没有在其他版本上测试  

## 如何使用？
不可单独使用，此为websocket客户端，需要连接至websocket服务器  
搭配用于QQ与Minecraft通讯的Astrbot插件可见 https://github.com/WhiteCloudOL/astrbot_plugin_mcqlink
